- 风险点 & 管控措施
- 产品交互示意 + Coze Demo 链接

## 🚀 部署到 GitHub Pages

### 方式一：直接上传（最简单）

1. 在 GitHub 新建一个仓库，如 `wereads-ai-prd`
2. 上传 `index.html` 文件
3. 进入仓库 Settings → Pages → Source 选择 `main` 分支 `/ (root)`
4. 保存后等待约 1 分钟，即可访问 `https://<你的用户名>.github.io/wereads-ai-prd/`

### 方式二：git 命令行

```bash
git init
git add .
git commit -m "feat: 微信读书 AI 推书助手 PRD 展示页"
git branch -M main
git remote add origin https://github.com/<你的用户名>/wereads-ai-prd.git
git push -u origin main
```

然后在 GitHub 仓库 Settings → Pages 开启即可。
